◆ COMM_RECV_BUFFER_SIZE

#define COMM_RECV_BUFFER_SIZE   1024

Size of the packet buffer for reading incoming packets by communication_readPackets().

The buffer must be large enough to contain a full packet excluding escape characters and packet delimiter character. That is, two bytes for packet length, one byte for channel and header checksum, and another byte for the checksum as well as the size of the payload. In total: size of payload + 4 bytes.

Maximum size is theoretically 65535, however it is also limited by the amount of available SRAM (= 8KB - static memory - dynamic memory - stack!)
