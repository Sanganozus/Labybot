18 /**
19  * Send a command on communication channel #CH_OUT_RDP to a remote data
20  * processor (for example MATLAB, Octave, Scilab) through HWPCS. The commands
21  * are added to a list (a script to be executed by a configured data processor)
22  * within HWPCS as long as the parameter execute is false. In order to add
23  * the last command and execute the whole script, set the parameter execute
24  * to true.
25  *
26  * Note that an internal buffer of 514 bytes needs to be temporarily
27  * allocated during this function.
28  *
29  * @param execute false to just add the command to the list,
30  * true to add the command and execute the whole script
31  * @param command a string containing the command, for format please see
32  * <a href="https://www.nongnu.org/avr-libc/user-manual/group__avr__stdio.html" target="_blank">vfprintf</a>
33  * from AVR Libc.
34  * The resulting string will be truncated after 512 characters
35  * excluding null termination.
36  * @return false if no memory for an internal buffer could be allocated, true on success
37  */
38 bool remoteDataProcessing_command(const bool execute, const char* command, ...);
