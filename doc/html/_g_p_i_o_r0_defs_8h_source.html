11 /**
12  * Definition of bit in GPIOR0 used by #TIMETASK macro and ISR
13  * <code>TIMER5_COMPA_vect</code>.
14  * The bit defined by #GPIOR0_INDEX_BIT serves an array index (which can only
15  * be 0 or 1) into an internally managed array timeTask_uptime[2]. This array,
16  * keeping the current and the previous uptime, and the index bit are written
17  * asynchronously in interrupt context. This information is used by the
18  * #TIMETASK macro when reading the current uptime in milliseconds. With this
19  * double buffering, there is no need to disable interrupts in main context
20  * when reading from the array, as access to the array index is atomic.
21  */
22 #define GPIOR0_INDEX_BIT 0
