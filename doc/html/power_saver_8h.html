Detailed Description

Power saving features for the ATmega1280.

Power is saved by disabling all of the ATmega1280 features. If a feature is required, it needs to be enabled explicitly by clearing the corresponding bit in either PRR0 or PRR1. See the ATmega1280's datasheet for details.

Note that the call to powerSaver_init() must be the first call in main()! This ensures that no feature gets disabled which otherwise may have been initialized before.
