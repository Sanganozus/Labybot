1 Overview {#mainpage}
2 ========
3 
4 HWPRobot provides both a library for often used functions in the HWP and a sample main.c file enabling
5 quick startup in programming the microcontroller and interfacing it to the HWP control system (HWPCS).
6 It also contains project files for Eclipse CDT as well as Atmel Studio with completely pre-configured
7 compiler and linker settings. The relevant files for Eclipse CDT are: .cproject and .project. Atmel
8 Studio uses the files: HWPRobot.atsln, HWPRobot.componentinfo.xml and HWPRobot.cproj. The project files
9 for Code::Blocks are: HWPRobot_Windows.cbp or HWPRobot_LinuxMac.cbp, depending on the OS.
10 
11 The directory @ref lib contains the library part with the contents as follows:
12 - @ref communication
13  + [communication.h](@ref communication.h): Functions for bi-directional communication with HWPCS
14  + [packetTypes.h](@ref packetTypes.h): Definition of communication packet types
15 - @ref io
16  + [adc.h](@ref adc.h): Analog to digital converter library
17  + [i2c.h](@ref i2c.h): Sending and receiving of I2C messages
18  + [led.h](@ref led.h): Access the user LED on the HWP board
19  + [uart.h](@ref uart.h): UART library: controls UART 0,1,2,3 via interrupts and FIFO buffers
20 - @ref motor
21  + [motor.h](@ref motor.h): Motor control functions
22 - @ref pathFollower
23  + [pathFollower.h](@ref pathFollower.h): Implementation of a path follower which interfaces with HWPCS
24 - @ref tools
25  + [fifo.h](@ref fifo.h): Simple implementation of a FIFO buffer using a ring buffer (circular buffer)
26  + [labyrinth.h](@ref labyrinth.h): Storage and manipulation functions for labyrinth walls
27  + [remoteDataProcessing.h](@ref remoteDataProcessing.h): Data and command transfer to a remote data processor (such as MATLAB, Octave, Scilab) for executing commands through HWPCS.
28  + [timeTask.h](@ref timeTask.h): Functions for timed execution of code and measurement of execution time
29  + [powerSaver.h](@ref powerSaver.h): Power saving features for the ATmega1280
30  + [variablesAccess.h](@ref variablesAccess.h): Provision of bitset data types and access to memory locations as different data types
31 
32 The directory @ref src contains the main file and the user configuration of the UART and ADC library:
33 - [main.c](@ref main.c): sample main file with rudimentary functionalities
34 - @ref cfg
35  + [adc_cfg.h](@ref adc_cfg.h): User configuration for ADC library
36  + [uart_cfg.h](@ref uart_cfg.h): User configuration for UART and communication library
